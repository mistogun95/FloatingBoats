creare il Per database aprire il server su una porta a piacere (tra quelle disponibili) e copiare nella barra degli indirizzi del browser il seguente path:
/db/setup.php
Per esempio: localhost:5000/db/setup.php

Per cancellare il database seguire invece il seguente path:
/db/clean.php
Per esempio: localhost:5000/db/cleanup.php

Ricordarsi di includere nella cartella db il file mysql_credentials.php per
rispettare il path di inclusione e scrivere all'interno le proprie credenziali 
d'accesso. Nella cartella UserDeploy sono presenti gli script php per la creazione
di database, tabella e inserimento dei dati utenti, nella cartella CleanUpDatabase
sono invece presenti gli script per l'eliminazione del database e delle tabelle.
